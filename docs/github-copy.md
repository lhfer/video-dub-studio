## 5) 发布帖文案（中文）
我开源了一个自己高频在用的工具：**Video Dub Studio**。

核心痛点是：看英文访谈时，不得不一直盯字幕，没法像播客一样后台听。

现在只要贴一个 YouTube 链接，就能自动完成：
1. 语音识别（Qwen ASR）
2. 高质量翻译（qwen3-max）
3. 多角色语音合成（Qwen TTS）
4. 时间轴对齐并导出新音轨/新视频

特别适合：长访谈、课程、行业播客。

欢迎试用和提 Issue，觉得有价值请给个 Star。

## 6) Launch 帖文案（英文）
I just open-sourced **Video Dub Studio**.

It solves a very specific pain: when watching long interviews in another language, you often need to stare at subtitles instead of listening in the background like a podcast.

With one YouTube URL, it runs a full Qwen pipeline:
1. ASR
2. Context-aware translation
3. Multi-speaker TTS
4. Timeline-aligned audio/video export

Great for long-form interviews, lectures, and research talks.
If useful, I'd appreciate a star and feedback.
